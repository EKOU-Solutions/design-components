# DataTable

An Excel-like data grid for accounting and tabular workflows. Supports keyboard navigation (ARIA grid pattern), inline cell editing, auto-computed totals, row creation/deletion, and disabled rows.

---

## Architecture

```
DataTable
├── Toolbar              — "+ Add row" button (shown when onAddRow is provided)
├── scroll wrapper       — overflow: auto, bounded by height prop or fill-parent
│   └── table[role=grid]
│       ├── thead        — ColumnHeader × N  +  optional Actions th
│       └── tbody        — DataRow × N  →  Cell | CellEditor | SelectEditor
└── totals wrapper       — width: 100%, always fills the full container
    └── table            — table-layout: fixed, TotalsRow
```

**Key decisions:**

- **Data-down, events-up.** DataTable owns no data. All mutations are signalled via callbacks (`onCellChange`, `onAddRow`, `onDeleteRow`). The parent controls state.
- **Two-table layout for totals.** The totals footer lives outside the scroll container so it is always visible and always fills 100% of the container width.
- **ARIA grid pattern.** The table has `role="grid"` and `aria-activedescendant`. Cells are identified by `id="cell-{rowIndex}-{colIndex}"`.
- **CSS custom properties for theming.** All tokens are declared in `tokens.css` on `:root` and can be overridden on any ancestor element.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `columns` | `ColumnDef[]` | — | Column definitions |
| `rows` | `RowData[]` | — | Row data — each row needs a unique `id: string` |
| `height` | `number \| "fill"` | — | Fixed max-height in px, or `"fill"` to stretch to parent |
| `className` | `string` | — | Extra class on root element |
| `initialFocusedCell` | `{ rowIndex, colIndex }` | — | Pre-focus a cell without user interaction |
| `onCellChange` | `(rowId, colKey, value) => void` | — | Called when a cell value is committed |
| `onAddRow` | `() => void` | — | Shows the + toolbar button; called on click |
| `deletable` | `boolean` | `false` | Adds a delete column with a trash icon per row |
| `onDeleteRow` | `(rowId) => void` | — | Called when the trash icon is clicked |
| `getRowDisabled` | `(row: RowData) => boolean` | — | Return `true` to make a row read-only |

### ColumnDef

| Field | Type | Description |
|-------|------|-------------|
| `key` | `string` | Unique key — matches property names in `RowData` |
| `label` | `string` | Header text |
| `width` | `number` | Width in px (default: 100) |
| `align` | `'left' \| 'right' \| 'center'` | Cell text alignment |
| `type` | `'string' \| 'number' \| 'select'` | Controls editor and totals computation |
| `options` | `SelectOption[]` | Required when `type='select'` |
| `editable` | `boolean` | `false` disables edit mode for this column |
| `placeholder` | `string` | Shown in text editor when value is empty |
| `showTotal` | `boolean` | Include this column in the totals footer |
| `totalLabel` | `string` | Static label in totals cell (e.g. `"TOTAL:"`); if omitted, sums numbers |

## Keyboard Navigation

| Key | Action |
|-----|--------|
| Arrow keys | Move focused cell |
| `Home` / `End` | First / last column in current row |
| `Ctrl+Home` / `Ctrl+End` | First / last cell in the grid |
| `Enter` or `F2` | Enter edit mode |
| `Escape` | Cancel edit, return to navigation |
| `Tab` (in edit mode) | Commit and move to next column |
| `Enter` on delete column | Trigger `onDeleteRow` |

Disabled rows (via `getRowDisabled`) are skipped during arrow navigation and cannot enter edit mode.

---

## Totals Footer

Set `showTotal: true` on any column to display the totals row.

- `type: "number"` columns are summed. Commas are stripped before parsing — `"1,234.56"` becomes `1234.56`.
- `totalLabel` shows a static string instead of a sum (useful for labelling the row).
- All other columns show an empty cell.

The totals table is a separate element outside the scroll area and always fills 100% of the DataTable container width.

---

## Cell Editors

**Text / Number** (`type: "string"` or `type: "number"`) — renders an input. Commits on Enter or blur. Cancels on Escape.

**Select** (`type: "select"`) — renders a custom ARIA listbox. ArrowDown/Up navigate options, Enter commits, Escape cancels. Arrow keys do not propagate to the table scroll container.

---

## Theming

Override any CSS custom property on a parent element:

```css
.my-container {
  --table-bg: #1e1e1e;
  --table-border: #444;
  --table-header-bg: #2d2d2d;
  --table-header-color: #f0f0f0;
  --table-row-hover-bg: #2a2a2a;
  --table-selected-bg: #2c3e50;
  --table-totals-bg: #2d2d2d;
  --table-font-size: 12px;
}
```

All tokens are declared in `tokens.css`.
